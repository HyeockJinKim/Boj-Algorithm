#### 시간 제한
> 2 초
#### 메모리 제한
> 128 MB
### 문제 내용

n×m의 0, 1로 된 배열이 있다. 이 배열에서 1로 된 가장 큰 정사각형의 크기를 구하는 프로그램을 작성하시오.



0

0


0

0



위와 같은 예제에서는 가운데의 2×2 배열이 가장 큰 정사각형이다. 

### 입력

첫째 줄에 n, m(1 ≤ n, m ≤ 1,000)이 주어진다. 다음 n개의 줄에는 m개의 숫자로 배열이 주어진다.

### 출력

첫째 줄에 가장 큰 정사각형의 넓이를 출력한다.
